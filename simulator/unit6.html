Common Patterns
for loop

Use for counting a fixed number of iterations.

for (int i = 0; i < 4; i++) {
    motors[i].setPower(0);
}
while (!isStopRequested())

The correct LinearOpMode main loop pattern.

while (!isStopRequested()) {
    // your code here
    telemetry.update();
}
loop() method

The correct TeleOp main loop — SDK calls this for you.

@Override
public void loop() {
    // runs repeatedly
}
